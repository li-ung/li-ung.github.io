ssh搭建web

一、环境准备

参考文章：三大框架SSH搭建web项目实例

参考视频：eclipse版ssh框架学习

  • Eclipse
  • JDK 19
  • struct 2.5.22
  • spring 5.3.9
  • hibernate 5.8.0
  • tomcat 10.0.27

二、环境配置

1.Eclipse：

  • 文本编码

窗口（Window）–>首选项（Preferences）–>常规（General）–>工作空间（Workspace）–>

文本文件编码（Text file encoding）选择其他（UTF-8）

  • 编辑辅助

窗口（Window）–>首选项（Preferences）–>Java–>编辑器（Editor）–>内容辅助（Content Assist）–>自动激活（Auto Activation）

在.的后面把键盘上每个字母都打一遍（看自身需求）

  • 配置Tomcat

    • 下载Tomcat

      https://tomcat.apache.org/

      选择要的版本

      往下划根据电脑选择相应版本下载

      解压到自己的路径中

      配置环境变量

      在系统变量中的path添加%CATALINA_HOME%in 和 %CATALINA_HOME%lib

      以管理员身份打开cmd，然后进入 tomcat 的 bin 目录，输入 service.bat install

      双击bin下的startup.bat文件,在浏览器输入localhost:8080即可看到tomcat界面

    • 配置

      窗口（Window）–>首选项（Preferences）–>Server–>Runtime Environment

      注意所选的版本需要跟刚刚下载的对应，否则报错

2.配置struct

  • 添加架包

5.单个架包

  • 下载

    https://mvnrepository.com/

  • 添加架包

    序号 其他架包 到Maven资源库中下载
    1 log4j-core-2.11.1.jar 日志要用到
    2 mysql-connector-java-5.1.26-bin.jar mysql驱动包
    3 aopalliance-1.0.jar aop配置事务时用
    4 aspectjrt-1.5.3.jar aop配置事务时用
    5 aspectjweaver-1.5.3.jar aop配置事务时用
    6 commons-dbcp-1.4.jar 使用jdbc.properties配置数据库时用
    7 commons-pool-1.6.jar 使用jdbc.properties配置数据库时用
